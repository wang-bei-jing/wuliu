研究、设计内容：
智能手机的普及使二维码技术应用于物流速递业务成为可能，可以充分发挥信息存贮量大，查询速度快、保护用户隐私等优点。
PC 版主要由物流公司的管理员使用，主要功能为：接收审核用户的订单；生成运单的二维码；订单，运单的管理；用户信息的反馈（留言板，增加，删除，更新，查询）；用户管理；员工管理等。
